# 1. 博客配置

# 基础配置

  • 全局安装

    npm install -g vuepress
    1

  • 创建博客目录

    mkdir vuepress-blog //创建博客文件夹
    1

  • 项目初始化

  • 初始化后会生成一个package.json文件

cd vuepress-blog //进入博客文件夹
npm init -y //初始化博客
1
2
  • 在当前目录中创建一个docs目录
mkdir docs //主要存放博客书籍内容
1
  • 首页内容书写(默认主题提供) README.md 博客首页文件
---
home: true
heroImage: /logo.jpg
actionText: 快速上手 →
actionLink: /zh/guide/
features:
- title: 简洁至上
  details: 以 Markdown 为中心的项目结构，以最少的配置帮助你专注于写作。
- title: Vue驱动
  details: 享受 Vue + webpack 的开发体验，在 Markdown 中使用 Vue 组件，同时可以使用 Vue 来开发自定义主题。
- title: 高性能
  details: VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。
footer: MIT Licensed | Copyright © 2018-present Evan You

# 核心配置

  • 在docs目录下创建.vuepress目录
cd docs
mkdir .vuepress //主要存放配置
1
2
  • 新建总配置文件config.js
cd .vuepress
touch config.js //config是整个项目的核心配置文件，所有菜单、导航栏相关的配置均配置在该模块中
1
2

# 2. 目录结构：

- vuepress-blog
	- docs 
		-.vuepress //配置博客的文件
        	- public //存放博客的静态资源的：如图片，css,js文件
            - config.js //配置网站的侧边栏，头部导航条； 博客的总配置文件
			- nav.js //配置头部导航栏
			- sidebar.js //配置导航侧边栏
        - READM.md //博客首页文件
	-package.json//npm init 初始化项目
